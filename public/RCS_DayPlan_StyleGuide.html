Lesson Flow Table Required
Four-column table: TIME | PHASE | ACTIVITY | PURPOSE. Appears under "Lesson Flow" section heading.
TIMEPHASEACTIVITYPURPOSE
0–5 min Settle Take attendance as students open their devices and get ready. Smooth start
5–35 min Watch Students open Microsoft Teams and navigate to TS Teaching Videos channel. Watch Session 5 video independently. Engage with content
If time remains Script activity Only once both video AND notes are fully complete. Students form groups of 3–4 and write a short script. Creative consolidation
Column widths: TIME 1300, PHASE 1600, ACTIVITY 4060, PURPOSE 2400 DXA. PHASE column: bold, BLUE. PURPOSE column: italic, #444444. Activity supports multiple lines. Alternating #F5F8FC / white rows.

3-column variant (no PURPOSE column, used for Worship Leadership): TIME 1300, PHASE 1800, ACTIVITY 6260.
What to Do If… Table Required
Scenario / response pairs. Always the last section before the final spacer.
…a student can't find the video:Remind them to check Teams > TS Teaching Videos > Session 5.
…a student rushes to the script:Check that both the video and Q5 notes are fully done first.
…something urgent arises:Contact the office immediately.
No header row. Column 1 (3000 DXA): gold left border (3px solid #C9973A), bold, #665500. Column 2 (6360 DXA): regular, #333333. Alternating #FFF3CD / white rows. Scenario text always begins with "…".

Section Order
Every lesson plan document follows this structure in this exact order.
1
Title Block
Logo + class name + subtitle + block/room/date/time detail line. Always navy background.
2
Spacer (200)
Standard gap after title block.
3
Info Box — Note to the TOC
3–5 bullet lines summarising what the TOC needs to know before anything else.
4
Optional insert Conditional
Step Panel, Leaders Box, or Activity Options Box — only when applicable to the class.
5
Section: Class Overview + KV Table
Class, Room, Time, Phones, What Comes Next — and any class-specific rows.
6
Section: Division of Roles + Roles Table
WHO / RESPONSIBILITY pairs. Always includes the TOC row first.
7
Section: Lesson Flow + Flow Table
TIME / PHASE / ACTIVITY / PURPOSE. Use 3-column variant if PURPOSE is not applicable.
8
Section: What to Do If… + What-If Table
3–6 scenario/response pairs. Always the last section. Final entry is always the urgent/office/Teams contact.
✓
Spacer (200)
End of document.
